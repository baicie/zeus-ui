# @zeus-web/textarea

Headless textarea primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/textarea
```

## React

```tsx
import { Textarea } from '@zeus-web/textarea/react'
```

## Vue

```vue
<script setup lang="ts">
import { Textarea } from '@zeus-web/textarea/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/textarea/wc'
```

```html
<zw-textarea>...</zw-textarea>
```
