# @zeus-web/input

Headless input primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/input
```

## React

```tsx
import { Input } from '@zeus-web/input/react'
```

## Vue

```vue
<script setup lang="ts">
import { Input } from '@zeus-web/input/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/input/wc'
```

```html
<zw-input>...</zw-input>
```
