# @zeus-web/card

Headless card primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/card
```

## React

```tsx
import { Card } from '@zeus-web/card/react'
```

## Vue

```vue
<script setup lang="ts">
import { Card } from '@zeus-web/card/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/card/wc'
```

```html
<zw-card>...</zw-card>
```
