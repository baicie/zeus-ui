# @zeus-web/radio-group

Headless radio-group primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/radio-group
```

## React

```tsx
import { RadioGroup } from '@zeus-web/radio-group/react'
```

## Vue

```vue
<script setup lang="ts">
import { RadioGroup } from '@zeus-web/radio-group/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/radio-group/wc'
```

```html
<zw-radio-group>...</zw-radio-group>
```
