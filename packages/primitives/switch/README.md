# @zeus-web/switch

Headless switch primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/switch
```

## React

```tsx
import { Switch } from '@zeus-web/switch/react'
```

## Vue

```vue
<script setup lang="ts">
import { Switch } from '@zeus-web/switch/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/switch/wc'
```

```html
<zw-switch>...</zw-switch>
```
