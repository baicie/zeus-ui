# @zeus-web/tooltip

Headless tooltip primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/tooltip
```

## React

```tsx
import { Tooltip } from '@zeus-web/tooltip/react'
```

## Vue

```vue
<script setup lang="ts">
import { Tooltip } from '@zeus-web/tooltip/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/tooltip/wc'
```

```html
<zw-tooltip>...</zw-tooltip>
```
