# @zeus-web/tabs

Headless tabs primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/tabs
```

## React

```tsx
import { Tabs } from '@zeus-web/tabs/react'
```

## Vue

```vue
<script setup lang="ts">
import { Tabs } from '@zeus-web/tabs/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/tabs/wc'
```

```html
<zw-tabs>...</zw-tabs>
```
