# @zeus-web/button

Headless button primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/button
```

## React

```tsx
import { Button } from '@zeus-web/button/react'
```

## Vue

```vue
<script setup lang="ts">
import { Button } from '@zeus-web/button/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/button/wc'
```

```html
<zw-button>...</zw-button>
```
