# @zeus-web/badge

Headless badge primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/badge
```

## React

```tsx
import { Badge } from '@zeus-web/badge/react'
```

## Vue

```vue
<script setup lang="ts">
import { Badge } from '@zeus-web/badge/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/badge/wc'
```

```html
<zw-badge>...</zw-badge>
```
