# @zeus-web/vue

Vue wrappers for Zeus Web primitives.

## Installation

```bash
pnpm add @zeus-web/vue
```

## Usage

```vue
<script setup lang="ts">
import { Button } from '@zeus-web/vue/button'
import { Input } from '@zeus-web/vue/input'
</script>

<template>
  <Button variant="primary">Save</Button>
  <Input placeholder="Email" />
</template>
```

## Components

Provides Vue 3 wrappers for all Zeus Web primitive components.
