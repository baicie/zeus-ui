# @zeus-web/collapsible

Headless collapsible primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/collapsible
```

## React

```tsx
import { Collapsible } from '@zeus-web/collapsible/react'
```

## Vue

```vue
<script setup lang="ts">
import { Collapsible } from '@zeus-web/collapsible/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/collapsible/wc'
```

```html
<zw-collapsible>...</zw-collapsible>
```
