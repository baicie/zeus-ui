# @zeus-web/dialog

Headless dialog primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/dialog
```

## React

```tsx
import { Dialog } from '@zeus-web/dialog/react'
```

## Vue

```vue
<script setup lang="ts">
import { Dialog } from '@zeus-web/dialog/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/dialog/wc'
```

```html
<zw-dialog>...</zw-dialog>
```
