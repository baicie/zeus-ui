# @zeus-web/select

Headless select primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/select
```

## React

```tsx
import { Select } from '@zeus-web/select/react'
```

## Vue

```vue
<script setup lang="ts">
import { Select } from '@zeus-web/select/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/select/wc'
```

```html
<zw-select>...</zw-select>
```
