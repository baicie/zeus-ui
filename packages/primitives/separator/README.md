# @zeus-web/separator

Headless separator primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/separator
```

## React

```tsx
import { Separator } from '@zeus-web/separator/react'
```

## Vue

```vue
<script setup lang="ts">
import { Separator } from '@zeus-web/separator/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/separator/wc'
```

```html
<zw-separator>...</zw-separator>
```
