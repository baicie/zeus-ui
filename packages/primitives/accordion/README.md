# @zeus-web/accordion

Headless accordion primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/accordion
```

## React

```tsx
import { Accordion } from '@zeus-web/accordion/react'
```

## Vue

```vue
<script setup lang="ts">
import { Accordion } from '@zeus-web/accordion/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/accordion/wc'
```

```html
<zw-accordion>...</zw-accordion>
```
