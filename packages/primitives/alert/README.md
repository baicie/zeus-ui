# @zeus-web/alert

Headless alert primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/alert
```

## React

```tsx
import { Alert } from '@zeus-web/alert/react'
```

## Vue

```vue
<script setup lang="ts">
import { Alert } from '@zeus-web/alert/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/alert/wc'
```

```html
<zw-alert>...</zw-alert>
```
