# @zeus-web/label

Headless label primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/label
```

## React

```tsx
import { Label } from '@zeus-web/label/react'
```

## Vue

```vue
<script setup lang="ts">
import { Label } from '@zeus-web/label/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/label/wc'
```

```html
<zw-label>...</zw-label>
```
