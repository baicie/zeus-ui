# @zeus-web/progress

Headless progress primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/progress
```

## React

```tsx
import { Progress } from '@zeus-web/progress/react'
```

## Vue

```vue
<script setup lang="ts">
import { Progress } from '@zeus-web/progress/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/progress/wc'
```

```html
<zw-progress>...</zw-progress>
```
