# @zeus-web/headless

Aggregated headless Web Components for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/headless
```

## Usage

```ts
import '@zeus-web/headless'
```

```html
<zw-button>Save</zw-button>
<zw-input placeholder="Email"></zw-input>
```

## Components

Aggregates all Zeus Web headless primitives into a single import.
