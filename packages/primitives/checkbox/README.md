# @zeus-web/checkbox

Headless checkbox primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/checkbox
```

## React

```tsx
import { Checkbox } from '@zeus-web/checkbox/react'
```

## Vue

```vue
<script setup lang="ts">
import { Checkbox } from '@zeus-web/checkbox/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/checkbox/wc'
```

```html
<zw-checkbox>...</zw-checkbox>
```
