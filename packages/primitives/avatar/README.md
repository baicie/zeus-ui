# @zeus-web/avatar

Headless avatar primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/avatar
```

## React

```tsx
import { Avatar } from '@zeus-web/avatar/react'
```

## Vue

```vue
<script setup lang="ts">
import { Avatar } from '@zeus-web/avatar/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/avatar/wc'
```

```html
<zw-avatar>...</zw-avatar>
```
