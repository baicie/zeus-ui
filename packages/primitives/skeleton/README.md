# @zeus-web/skeleton

Headless skeleton primitive for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/skeleton
```

## React

```tsx
import { Skeleton } from '@zeus-web/skeleton/react'
```

## Vue

```vue
<script setup lang="ts">
import { Skeleton } from '@zeus-web/skeleton/vue'
</script>
```

## Web Component

```ts
import '@zeus-web/skeleton/wc'
```

```html
<zw-skeleton>...</zw-skeleton>
```
