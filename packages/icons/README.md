# @zeus-web/icons

Multi-framework icon set for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/icons
```

## Usage

### React

```tsx
import { IconArrowRight } from '@zeus-web/icons/react'
```

### Vue

```vue
<script setup lang="ts">
import { IconArrowRight } from '@zeus-web/icons/vue'
</script>
```

### SVG

```ts
import '@zeus-web/icons/svg/arrow-right'
```
