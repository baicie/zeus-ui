# @zeus-web/themes

Theme tokens for Zeus Web.

## Installation

```bash
pnpm add @zeus-web/themes
```

## Usage

### CSS

```css
@import '@zeus-web/themes/default.css';
```

### JavaScript

```ts
import '@zeus-web/themes/default.css'
```

## Themes

Provides design tokens and component-level CSS variables for Zeus Web.
